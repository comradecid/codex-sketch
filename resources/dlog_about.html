This plugin is used to sync changes with UI components represented in Sketch libraries with your Codex style guide.

Set it up first

After installing the plugin, you'll need to first configure it for use. In the Sketch app menu, go to Plugins > Codex > Preferences; you should then see the Preferences dialog. The first, most important step is to sign in; without this, the style guide won't accept any of your changes. Click on Sign in, and enter your username and password.

Next, you should take note of the 'Ignore items whose names begin with...' section; these are settings for how to tell the plugin to 'ignore' specific items, so that no presentation changes to those items are sent to the style guide (even if you've selected them to be synced). Usually you won't have to change anything in this section. Close the Preferences dialog and your settings will be saved.

Syncing with the style guide

Select those components in your Sketch file that you'd like to update in the style guide.
